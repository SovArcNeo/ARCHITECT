## ‚ö° System Capabilities

### üß† The Neural Core
ARCHITECT utilizes a sophisticated, weighted-voting ensemble method (30% / 40% / 30%) combining three distinct models:
1.  **MLP Classifier:** Sequence analysis (128/64/32 neurons).
2.  **MLP Regressor:** Anomaly detection via autoencoder bottleneck (16/8/16 neurons).
3.  **Pure NumPy Network:** Pattern recognition built from scratch without heavy frameworks.

### üñ•Ô∏è High-Fidelity GUI
* **Modular 4-Part Architecture:** Components can be detached and moved across screens.
* **Custom Tkinter Engineering:** Heavily modified widgets for a "Sci-Fi" terminal aesthetic.
* **Live Data Streaming:** Matrix-style waterfall text for raw data packet visualization.

### üõ°Ô∏è Operational Security

* **Minimal Footprint:** Bloat-free design ensuring high performance on standard Linux hardware.
